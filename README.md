# ✈️ Travel Journal

> A travel website featuring destinations, itineraries, and travel tips across India and the world.

---

## 🌍 About the Project

Travel Journal is a front-end web project built to document and showcase beautiful travel destinations across India and beyond. Whether you are planning a weekend trip to Manali or a grand tour of Rajasthan, this project has something for every traveller.

---

## 🗺️ Featured Destinations

| Destination | State | Best Time to Visit |
|---|---|---|
| Manali | Himachal Pradesh | October – February |
| Goa | Goa | November – February |
| Jaipur | Rajasthan | October – March |
| Ladakh | J&K | June – September |
| Kerala Backwaters | Kerala | September – March |
| Coorg | Karnataka | October – April |

---

## 🚀 Features

- Homepage with hero banner and destination highlights
- Destination cards with photos, descriptions, and best time to visit
- Trip itinerary planner section
- Budget travel tips
- Responsive design for mobile and desktop
- Contact / booking form

---

## 🛠️ Tech Stack

- **HTML5** – structure
- **CSS3** – styling and layout
- **JavaScript** – interactivity
- **Google Fonts** – typography

## 🧭 How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/Anushka-4848/travel-journal.git
   ```
2. Open `index.html` in your browser.
3. That's it — no server needed!
